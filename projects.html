Projects

During my programming journey, I've embarked on numerous projects, each offering valuable insights. Among the multitude of scripts I've crafted, several noteworthy ones stand out. These include projects that challenged me to innovate, problem-solve, and refine my coding skills. Each endeavour has contributed to my growth as a programmer, broadening my understanding and proficiency in various domains of software development. Through these projects, I've honed my ability to conceptualise ideas, implement solutions, and adapt to evolving technologies. They serve as milestones in my journey, reflecting my dedication to continuous learning and exploration within the realm of programming.

Forest fire analysis using satellite data

This task involves accessing and processing satellite data in MATLAB. The MERRA2 data base provides a range of atmospheric observations, hosted by NASA, and is created through several different satellite observations that are then post-processed to provide continuous coverage of the planet. During this project I worked on cleaning and plotting the data onto a suitable geographical map and creating an animation of a series of plots to fully visualise the atmospheric dynamics.

Planetary orbit simulator

I developed a script in python using Matplotlib.pyplot to produce animations of planetary orbitals to assist me with my EPQ. This was extended to work with multiple planetary bodies by using Fourier series, and traced out each object’s trajectory over time. This allowed me to calculate unique trajectories, such as triangular or square orbital paths.

Handwritten number recognition

This project involved an introductory exposure to the neural network class of machine learning algorithms and to test how well they perform against known data. Initially, it utilised a dataset from Sklearn consisting of 8x8 pixel handwritten numbers. Subsequently, the project extended its scope to handle any handwritten number input. Through the implementation of neural networks, the system achieved a classification accuracy of over 96%. The project also enhanced my proficiency in data handling techniques.

Quantum tunneling calculator

The project employed the Schrödinger equation to calculate the probability of an arbitrary object quantum tunnelling through a barrier. While not having any practical applications (as probabilities are extremely unlikely), it was a fun project to push my mathematical capabilities to their limits. The script allowed the user to enter the dimensions and atomic composition of the object, as well as the thickness of a wall (and or potential barrier), and find the probability of the object appearing on the other side.

Flappy bird in python

I developed a Flappy Bird game using Pygame, marking one of my initial encounters with this library. This project enabled me to explore game mechanics, sprite management, and collision detection. These skills have extended to other projects that I have undertaken, like reproducing my own version of geometry dash and Pac-man.

DurHack web dev

Engaged in the Durham 24-hour hackathon, where a friend and I crafted a space-themed e-commerce platform inspired by Tinder's concept. It involved integrating product data from THG and employing HTML, CSS, JavaScript, and GO for the website hosting. I primarily focused on front-end development, with occasional contributions to the back-end, while my partner dedicated efforts to the back-end aspects. We were then responsible for presenting our project to a selection of judges.

Matlab truss structure analysis

During this project, I aimed to create the lightest possible structure that can support a load placed on a variety of different nodes of a truss (a rigid structure composed of connected elements) while also complying to a set of constraints, like maximum deflection and cross-sectional area. Additionally, I planned to use the data collected to compare the calculated values using two different optimisers and evaluate each optimisers’ effectiveness’. The two optimisers which I evaluated were MATLAB’s fmincon and simulannealbnd.

iOS calculator app

Using SwiftUI, I embarked on creating a rudimentary yet functional calculator application. It provided me with valuable insights into mobile app development paradigms and equipped me with foundational knowledge to undertake more ambitious projects in the future. Through iterative design and testing, I honed my skills in user interface design and logic implementation.
